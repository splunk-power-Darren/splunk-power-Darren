## Visualization Basics

### What Visualizations Do
- Turn search results into charts, tables, and dashboards  
- Help spot trends, patterns, and anomalies  
- Used heavily in SOC dashboards and monitoring

---

## Statistics Tables
- Displays results in a clean, column‑based layout  
- Great for validating data before building charts  
- Works with `stats`, `chart`, and `timechart`

## Single Value Visualizations
- Shows one key metric (count, avg, max, etc.)  
- Useful for KPIs like failed logins, errors, or alerts

## When to Use Each
- **Statistics Table:** validating data  
- **Single Value:** KPIs / summary metrics  
- **Event Listing:** raw logs / investigation  
- **Charts:** trends, comparisons, patterns
