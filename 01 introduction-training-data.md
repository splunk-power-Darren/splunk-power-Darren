# Module 1 – Introduction and Training Data

## Video 1 – About Splunk Core Cert Power User Course

## Purpose of the Power User Course
The Power User course builds on the Splunk Core User prerequisite and focuses on developing practical SPL skills, field usage, transforming commands, lookups, and dashboards.

## Why the Course Is Valuable
Splunk Enterprise is expensive and not easily accessible for individuals. This course provides a structured environment and training dataset to practice without needing a full Splunk license.

## Certification Path
- Splunk Core User (prerequisite)
- Splunk Power User (this course)
- Splunk Enterprise Certified Admin

## Course Structure Overview
The course progresses through search fundamentals, fields, transforming commands, lookups, dashboards, and exam preparation.

## Video 2 – Course and Exam Prerequisites

## Overview
In this video, the instructor explains how the Splunk Power User course is organized and provides an overview of the modules included in the training.

## Module Breakdown
Below are the corrected modules covered in the course:

### Module 1 – Introduction and Training Data
Covers the purpose of the course, prerequisites, and the dataset used throughout the training.

### Module 2 – Transforming Commands for Visualizations
Focuses on using transforming commands to summarize data and prepare it for visualizations.

### Module 3 – Advanced Visualizations
Covers more complex visualization options and how to build advanced charts in Splunk.

### Module 4 – Filtering and Formatting Results
Explains how to refine search results, apply filters, and format output for clarity.

### Module 5 – Correlating Events
Shows how to combine multiple data sources or event types to identify relationships and patterns.

### Module 6 – Creating and Managing Fields
Covers field extractions, field creation, and managing fields within Splunk.

### Module 7 – Creating Field Aliases and Calculated Fields
Explains how to create field aliases and calculated fields to enhance data usability.

### Module 8 – Creating Tags and Event Types
Covers how to classify events using tags and event types to support organization and faster searching.

### Module 9 – Creating and Using Macros
Introduces macros and how to use them to simplify complex SPL searches.

### Module 10 – Creating and Using Workflow Actions
Shows how to create custom workflow actions to enhance investigation workflows.

### Module 11 – Creating Data Models
Covers the basics of building data models to support structured analysis and pivoting.

### Module 12 – Using the Common Information Model (CIM) Add-on
Explains how to use CIM to normalize data and support correlation across different data sources.

This video provides a roadmap for the entire course and explains what each module will focus on.




## Video 3 – Course Outline

## Overview
In this video, the instructor provides a detailed outline of how the Splunk Power User course is structured. This includes the flow of the training, how each module builds on the previous one, and what learners should expect as they progress.

## Course Outline Summary
The instructor explains the overall learning path and how the course is organized:

- The course begins with foundational concepts such as understanding the training data and basic search behavior.
- It then moves into transforming commands and visualizations to help learners understand how to summarize and present data.
- More advanced topics follow, including filtering, formatting, correlating events, and managing fields.
- Later modules introduce knowledge objects such as tags, event types, macros, workflow actions, and data models.
- The course concludes with the Common Information Model (CIM) add-on and how it supports data normalization.

## Purpose of the Outline
The goal of this video is to give learners a clear roadmap of the course so they understand:

- What topics will be covered
- How the modules connect to each other
- How the skills build toward the Splunk Power User certification

This video sets expectations for the entire course and shows how each module contributes to building practical, real-world Splunk skills.
## Video 4 – Add Training Data – Overview

## Overview
In this video, the instructor walks through how to add the provided `training_data.zip` file into Splunk. He explains what the ZIP file contains and why this dataset is used throughout the course.

## What the Video Covers

### Adding the Training Data
- The instructor shows where to download the `training_data.zip` file.
- He demonstrates how to upload the data into Splunk.
- The process includes selecting the ZIP file, choosing the correct source type settings, and completing the data input steps.

### What’s Inside the ZIP File
The instructor explains the contents of the training dataset, including:
- Multiple log files used for exercises throughout the course.
- Data representing different systems and event types.
- Files designed to support modules on fields, visualizations, correlations, macros, workflow actions, and data models.

### Purpose of the Dataset
- Provides a consistent environment for all students.
- Ensures everyone has the same logs for searches, visualizations, and knowledge object creation.
- Simulates a small but realistic Splunk environment for hands‑on practice.

## Key Takeaway
This video prepares learners for the rest of the course by ensuring the training data is properly loaded and understood. The dataset is used in every module that follows.
